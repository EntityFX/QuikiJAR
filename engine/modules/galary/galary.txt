ссылка будет выглядеть так:
site.com/galary/123/4556/65
123 - имя юзера, у которому принадлежит галерея.
4556 - название альбома.
65 - id фотки

количество параметров: 3
если есть 1 параметр - показать списки галерей пользователя (+ get-запрос переменной listnum:  ?l=1).
если есть 2 параметр - показать списки файлов в галерее.
если есть 3 параметр - показать отдельный элемент галереи.

Как же будет осуществляться загрузка файлов?
В первую очередь пользователь должен создать что? - правильно, альбом, в который будет добавляться картинка. Значит, нужно сперва сделать функцию, которая создаст альбом (в таблицу galary_list). Сразу же должны быть поля: имя альбома, комментарий, приватность (кто может, а кто не может видеть). В пустом альбоме обложка будет выбираться из условия - какой последний файл залит, тот и будет обложкой. Естесственно, потом можно будет назначить любой из файлов.
Итак, альбом создан в таблице. После этого загрузчику передаются данные: id альбома

д.б.:
1) уменьшение размера
2) автопереименование
3) проверка
