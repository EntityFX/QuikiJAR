Функции:
	$(selector) Возращает ссылку на обёрнутый набор, полученный по параметру selector.
Методы:
	.html(string) Меняет внутреннее содержимое элементов на значение в string
	.each(function()) Выполняет проход по обёрнутому набору. function() - функция вызова. Доступ к элементу через this
	.filter(selector) Выбирает из набра отсеянный набор по selector
	.size() Возращает количество элементов в обёрнутом наборе
	.get(index) Возращает объект из обёрнутого набора по индексу. Аналогично [index] - как с массивами. \Метод-индексатор
	.index(element) Возращает индекс по элементу, который содержится в обёрнутом наборе. element - ссылка на элемент в наборе
	.add(element|string|Array) Добавляет к существующему набору новые элементы из массива, из набора или из строки
	.addClass(className) Добавляет к элементам набора класс с именем className
	.append(element|string|Array) Добавлеят в конец содержимого каждого элемента новые элементы из массива, из набора или из строки
	.зкуpend(element|string|Array) Добавлеят в начало содержимого каждого элемента новые элементы из массива, из набора или из строки
	.slice(from,to) Возвращает срез из обёрнутого набора. from - начальный индекс, to - конечный индекс.
Свойства:
	.innerHTML Получает/Устанавливает содержимое обёрнутого набора
	.length Получает число элементов в обёрнутом наборе
